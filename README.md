# LgbmExx

LgbmExx is a library of ancillary functions that use [LgbmEx](https://github.com/tato-gh/lgbm_ex).

- `cross_validate/3`
- `grid_search/4`
- `map_importance_rate/2`
- `one_hot_encode/3`
  - utility
- `columns_stats/2`
  - utility
- `get_correlation_map/2`
  - utility

**NOTE**

- Beta version / Not stable

## Instration

```elixir
def deps do
  [
    {:lgbm_exx, "0.0.1", github: "tato-gh/lgbm_exx"}
  ]
end
```
